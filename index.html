⚡ Data & API

Connect NTDeliver data to your own systems — DHIS2, Power BI, Tableau, Metabase, or any custom dashboard. Role-based access ensures each organisation sees only its own data.

API Catalog

GET
Shipments API
/api/v1/shipments
POs, shipment lines, status, dates, quantities, donor, and pharma. Filterable by country, year, drug, and stage.
GET
Inventory API
/api/v1/inventory
On-hand quantities by batch, warehouse location, and expiry date.
GET
Planning & MDA API
/api/v1/mda-plans
Planned MDA rounds, target populations, quantities needed per drug and country.
GET
KPIs & Analytics API
/api/v1/kpis
Pre-aggregated indicators by country, year, and product — where agreements allow.

Quick-start recipes

📡 Get next 6 months of shipments for a country
GET /api/v1/shipments?country=NGA&after=2025-03-01&before=2025-09-01 Authorization: Bearer YOUR_API_KEY
📊 List MDA plans and compare to stock
GET /api/v1/mda-plans?country=NGA&year=2025 GET /api/v1/inventory?country=NGA # Join on drug code to compute adequacy gap
🔌 Connect to DHIS2 / Power BI
Use the Shipments and Inventory endpoints as data sources in Power BI's "Get Data → Web" connector, or schedule a daily DHIS2 import using the /api/v1/shipments feed. Column dictionaries available in the Downloads section.
Access & governance
Country users — own country data only
Donor/Partner — portfolio countries
Pharma — global or own products (per agreement)
